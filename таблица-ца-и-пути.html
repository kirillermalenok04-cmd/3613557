веб-приложения «D&P Perfumum»

Таблица 1. Портрет целевой аудитории

№ Сегмент ЦА Профиль Цели Потребности Функции сайта Ограничения
1 Гость 18–55 лет, пользователи интернета Оценить ассортимент и цены до регистрации Просмотр каталога, отзывов, оплаты и контактов без входа Разделы: Парфюмы, Отзывы, Оплата, Контакты Корзина и заказ только после авторизации
2 Покупатель 20–45 лет, онлайн-покупатели парфюмерии Выбрать аромат и оформить заказ Каталог, корзина, простой вход в систему Регистрация, вход, корзина, оформление заказа Оплата в текущей версии — уведомление, без платёжного шлюза
3 Постоянный клиент 25–50 лет, знаком с магазином Повторная покупка, ориентир на отзывы Быстрая навигация, доверие к отзывам Каталог, отзывы, контакты (телефон, e-mail) Нужна скорость работы интерфейса
4 Администратор Сотрудник / владелец магазина Контроль данных, отчётность Сводка и выгрузка информации Админ-панель, экспорт JSON Часть данных хранится в браузере (localStorage)

Таблица 2. Основные пользовательские пути

Путь Действующее лицо Этапы Результат
П1 Гость Главная → Каталог → Отзывы / Оплата / Контакты Решение о покупке или уход с сайта
П2 Гость → Покупатель Каталог → Вход / регистрация → Корзина → Оформление заказа Подтверждение заказа, очистка корзины
П3 Покупатель Вход → Каталог → Корзина → Оформление Повторная покупка без регистрации
П4 Гость / Покупатель Контакты → Телефон / E-mail / Карта Связь с магазином
П5 Администратор Вход (роль admin) → Админ-панель → Экспорт данных Файл JSON со сводными данными
