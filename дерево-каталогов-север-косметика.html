«СЕВЕР КОСМЕТИКА» — среда разработки (VS Code / Cursor)

Для скриншота в редакторе: раскройте папку prisma, папку node_modules оставьте свёрнутой.

├── 📁 prisma/
│ └── 📄 schema.prisma
├── 📁 node_modules/ (свёрнуто)
├── 📄 index.html
├── 📄 styles.css
├── 📄 script.js
├── 📄 server.js
├── 📄 package.json

├── 📄 .env.example
├── 📄 .gitignore
├── 📄 README.md
└── 📄 Макет парфюмюм.html
Каталог / файл Назначение
prisma/ Слой доступа к БД: описание моделей данных (User, Order, Review) и связей для ORM Prisma
node_modules/ Установленные npm-зависимости (Express, Prisma, bcrypt и др.); не редактируется вручную
index.html Главная страница магазина: разметка интерфейса, вкладки, модальные окна
styles.css Таблица стилей: оформление каталога, корзины, адаптивная вёрстка
script.js Клиентская логика: каталог, корзина, авторизация, отзывы
server.js Сервер Node.js: REST API, раздача статики, работа с PostgreSQL
package.json Метаданные проекта, список зависимостей и npm-скрипты запуска
railway.json Конфигурация развёртывания на облачной платформе Railway
.env.example Шаблон переменных окружения (DATABASE_URL и др.)
.gitignore Список файлов, исключаемых из системы контроля версий Git
README.md Документация: установка, запуск, деплой
